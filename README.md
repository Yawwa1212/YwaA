# GANG WON LAND (Vegas Wheel Build)

정적 파일 – GitHub Pages에 업로드 시 바로 동작.

## 룰렛(총 25칸)
- 20x: 1칸
- 10x: 3칸
- 3x: 5칸
- 1: 10칸 (당첨 시 2배 지급)
- -2x: 5칸 (베팅 총액 기준 추가 패널티)
- -10x: 1칸 (베팅 총액 기준 큰 패널티)
- BOOST: 1칸 (다음 스핀 결과 배수 2배, 연속으로 걸리면 2^n)

## 베팅
- 20 / 10 / 3 / 1(=2배) 에 각각 금액을 따로 입력.
- 스핀 시 총 베팅액이 먼저 차감되고,
  - 당첨 숫자에 걸린 금액만 배수로 지급(BOOST 적용)
  - -2x/-10x는 총 베팅액 기준 추가 차감
  - BOOST는 현금 변동 없이 '다음 결과 배수'만 증가

## 자금/대출
- 시작 자금: 30pen
- 최대 대출(빚): 100pen
- 현금이 0pen일 때만 대출 가능
- 1회 33pen씩, 마지막 1회는 34pen(총 100pen)
